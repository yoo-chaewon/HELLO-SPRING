>
> 2-1. Java 파일을 이용한 프로젝트 진행
>
> 2-2. 우선 따라해보는 스프링 프로젝트



#### 2-2. 우선 따라해보는 스프링 프로젝트

- applicationContext.xml

  ```xml
  <?xml version="1.0" encoding="UTF-8"?>
  <beans xmlns="http://www.springframework.org/schema/beans"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://www.springframework.org/schema/beans
          http://www.springframework.org/schema/beans/spring-beans.xsd">
  
      <bean id="tWalk" class="TransportationWalk" />
  
  </beans>
  ```

- 리소스 파일에서 모듈들 가져오기

  ```java
  public class MainClass {
      public static void main(String[] args){
  //        TransportationWalk transportationWalk = new TransportationWalk();
  //        transportationWalk.move();
  
          GenericXmlApplicationContext applicationContext = new GenericXmlApplicationContext("classpath:applicationContext.xml");
          TransportationWalk transportationWalk = applicationContext.getBean("tWalk", TransportationWalk.class);
          transportationWalk.move();
  
          applicationContext.close();
      }
  }
  ```
